Outer Elements

- Menu bar  
  - The Players cup trophy is the menu  
  - When tap, the menu will expand left and right  
  - On the expanded menu are the following options left to right  
    - Pin \- tapping this serves as way to get back to the map view   
    - Phoenix \- tapping this serves as a way to see the available offers in a swipeable card manner on top of the map  
    - Player cup trophy \- tapping this serves as a way to close the menu  
    - Leaderboard symbol \- tapping this serves as a way to access the leaderboard of collected GSky Token points  
    - Friend \- tapping this serves as a way to bring up a menu to invite friends and follow friends. You can also initiate a chat with them  
    - Chat symbol \- tapping this serves as a way to access and chat with friends  
- Home button \- tapping this serves as a way to recenter the map on the player character  
- Profile picture \- Uploaded profile picture has player level on top. Tapping these serves as a way to access Profile/Account page  
- Notification \- tapping this serves as a way to see notifications  
- AR Symbol \- tapping this serves as a way to enter AR mode

Player Movement

- Every 1200 steps taken, Load 4-12 more of GSKY token denominations on the map  
  - Previously loaded tokens do not disappear, until the user closes the app. Essentially new tokens are loading every time the player exits the game for 30mins or more before returning to the game   
  - User has to be within a 500 ft distance from the token to claim  
  - Tokens can load in a radius of up 2000 ft distance from the character  
  - When claimed, they disappear adding value to the player’s point balance
